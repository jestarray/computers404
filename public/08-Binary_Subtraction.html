Binary Subtraction

#How to subtract binary numbers?

x + y = x + -y


Subtraction is just addition with negative numbers! To put it shortly, if you already know how to add two binary
numbers, you know how to subtract.
In decimal for example:

12 - 4 = 12 + -4
8 = 8


Now let’s do this in binary.

  1. Subtracting via adding negative binary numbers
    12 in binary
    8421
    1100
    -4 in binary
    8421
    1100
    1 1
[ 1 1 0 0 ] (12)
+ [ 1 1 0 0 ] (-4)
_______________
1 [ 1 0 0 0 ] (8)
^ overflow is ignored because it can’t fit inside the bit width

    resulting in:

    8 in binary
    8421
    1000

    Due to tossing away the overflow, we get 10002, which is 810 in decimal and is the correct answer to
    12 - 4.

Extra Examples
Example: -8 + 3 = -5

[ 1 0 0 0 ] (-8)
+ [ 0 0 1 1 ] (3)
_______________
[ 1 0 1 1 ] (-5)


Example: -6 + -1 = -7

1 1 1
[ 1 0 1 0 ] (-6)
+ [ 1 1 1 1 ] (-1)
_______________
1 [ 1 0 0 1 ] (-7)
